# DOMAIN HEALTH PROFILE CONSTITUTION

- 건강정보는 회원 기본 프로필이 아니라 복약 안전 판단을 위한 사용자 건강 프로필로 취급한다.
- 건강 프로필은 마이페이지에서 입력하고 관리하되, 실제 활용 가치는 의약품 검색, 상세, 내 복용약 화면의 주의 안내 강화에 둔다.
- 1차 버전에서는 의료 판단 자동화보다 사용자가 먼저 점검해야 할 주의 신호를 명확히 보여주는 것을 우선한다.
- 건강 프로필이 비어 있어도 핵심 의약품 기능은 사용할 수 있어야 한다.
- 건강 프로필 항목은 사용자가 이해하기 쉬운 생활형 질문으로 제시하되, 복약 주의와 직접 연결되는 항목만 포함한다.
