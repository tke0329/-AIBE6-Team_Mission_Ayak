# DOMAIN HEALTH PROFILE STATUTE

- 건강 프로필은 `기본 상태`, `생활 습관`, `기저 질환`, `약물 안전` 4개 묶음으로 구성한다.
- `기본 상태`에는 나이, 임신 여부, 수유 여부를 포함한다.
- `생활 습관`에는 음주 빈도, 운전 여부를 포함한다.
- `기저 질환`에는 간 질환 여부, 신장 질환 여부, 고혈압 여부, 천식 여부, 위장 질환 여부를 포함한다.
- `약물 안전`에는 알레르기, 현재 복용 중인 약을 포함한다.
- 1차 버전의 연령 입력은 숫자 나이를 직접 받고, 내부적으로 `어린이`, `청소년`, `성인`, `고령자`로 자동 분류한다.
- 연령 분류 기준은 `0-12세 어린이`, `13-18세 청소년`, `19-58세 성인`, `59세 이상 고령자`다.
- 1차 버전의 음주 입력은 `거의 안 마심`, `월 1~3회`, `주 1~2회`, `주 3회 이상` 중 하나를 고르는 방식으로 관리한다.
- 불리언 성격의 항목은 자유 서술 대신 `예/아니오` 선택을 우선 사용한다.
- 알레르기와 현재 복용 중인 약은 자유 입력을 허용하되, 이후 내 복용약 데이터와 연결 가능한 구조로 확장 가능해야 한다.
- 마이페이지 건강 프로필 섹션은 데이터가 없을 때 `입력하기`, 데이터가 있을 때 핵심 요약과 `수정하기`를 보여준다.
- 마이페이지는 건강 프로필의 입력과 확인을 담당하고, 실제 주의 문구 노출은 의약품 검색 결과, 의약품 상세, 내 복용약 화면에서 재사용할 수 있어야 한다.
- 의약품 상세 상단에는 건강 프로필과 직접 맞닿는 주의 안내만 별도 요약으로 먼저 보여줄 수 있다.
- 내 복용약의 `통합 복용 주의`는 건강 프로필과 맞닿은 안내에 가산점을 부여해 상단으로 재정렬하되, 관련 없는 기존 안내를 숨기지는 않는다.
- 건강 프로필이 비어 있으면 경고 판단을 과장하지 말고, 프로필 입력 시 더 정확한 복약 주의를 확인할 수 있다는 안내만 보여준다.
- 별도 저장 스키마가 확정되기 전 1차 버전의 건강 프로필은 로그인 사용자 기준 브라우저 로컬 저장으로 관리한다.
